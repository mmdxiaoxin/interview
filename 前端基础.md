8. **响应式设计**：

10. **CSS 预处理器**：

11. **模块化和组件化**：
    - CSS 模块化的概念
    - CSS 组件化的实践
12. **性能优化**：
    - 减少选择器嵌套层级

13. **现代 CSS 技术**：
    - 自定义属性（CSS 变量）
